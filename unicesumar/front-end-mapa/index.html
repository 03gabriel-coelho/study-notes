Sobre Mim

Sou um Desenvolvedor Web Full Stack apaixonado por tecnologia e soluções inovadoras. Minha trajetória no desenvolvimento começou no front-end, trabalhando com CSS, Javascript e HTML puro em projetos para instituições de Belo Horizonte. Essa experiência foi fundamental para consolidar minha base técnica e entender as demandas do mercado. Com o tempo, evoluí para o desenvolvimento full stack, atuando com React.js, Node.js, e PHP, ganhando experiência tanto no front-end quanto no back-end. Atualmente, trabalho como desenvolvedor full stack, aprofundando meu conhecimento em infraestrutura, automação e servidores.

💡 Principais habilidades/competências:

✅ Desenvolvimento de aplicações escaláveis com React.js, TypeScript, Next.js e Node.

✅ Integração com bancos de dados Oracle PL/SQL e arquitetura back-end robusta.

✅ Implementação e manutenção de pipelines CI/CD com GitHub Actions, automatizando deploys e builds.

✅ Gerenciamento de servidores Linux físicos, criando um ambiente auto-hospedado para staging e produção.

✅ Criação de ambientes de homologação integrados ao fluxo de merge, permitindo testes antes do deploy.

✅ Colaboração direta com equipes multidisciplinares e aplicação de metodologias ágeis (Jira, ClickUp e Trello).
